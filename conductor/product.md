# Product Definition — Spool Tracker

## 产品概述

Spool Tracker 是面向 3D 打印用户的耗材管理系统，解决"料卷在哪"和"料卷是否已用完"两个高频问题。通过二维码将实物料卷、位置标签与 Web 数据绑定，支持移动端扫码更新和桌面端批量浏览。

## 核心功能

- 料卷生命周期管理（ACTIVE / EMPTY）
- 耗材字典管理（品牌、材料、颜色、打印参数）
- 位置管理（货架/打印机/AMS 插槽/干燥机/自定义）
- 扫码改位置（UPC/GTIN 扫码分配 + QR 码扫码）
- 标签打印（料卷标签 + 位置标签）
- 品牌 Logo 上传
- 移动端优先 + 桌面侧边导航

## 用户画像

个人 3D 打印爱好者，家庭实验室场景，单用户使用。

## 数据模型

```
GlobalFilament（耗材字典/SPU）
  └── Spool（料卷/SKU）
        └── Location（位置）
```

## 关键目标

1. 快速定位任意料卷的物理位置
2. 移动端扫码操作流畅（< 3 步完成位置更新）
3. 数据本地化，无需云服务
